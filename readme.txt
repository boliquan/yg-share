== Description ==

Important: The more infomation in [homepage](http://boliquan.com/yg-share/).

YG-Share can automatically insert the sharelist at the end of the article ,such as "Delicious".

It can almost work well at any version of WordPress.

YG-Share插件可以自动在文章末尾插入一个社会化分享列表，比如分享到“Delicious”。

== Installation ==

1. Download and extract `yg-share.zip` to `wp-content/plugins/`

2. Activate YG-Share in your Admin Panel
